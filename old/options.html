HERO Timesheet

Define the default value for Timesheet Record Description, using this format:
Text|Value|Default
One item in per line
Custom JS Scripts to be embed to the page, each script url per line
For example: //cdnjs.cloudflare.com/ajax/libs/underscore.js/1.9.1/underscore-min.js

You can do magic thing using callback functions, let's become a HERO timesheet!

Accessible varialbes: userId, csrf, myWorkingHoursByDate

Function can be called in onPageReady, onSubPageReady:

addTimesheetRecord(division, task, project, date, timesheethours, taskdescription = '', worklocation = 'Local', isdraft = true, callback = null) - Input value the same in add timesheet form

Don't use callback function if you don't know what are you doing, it can stop the extension ;)
onLoad callback function will be triggered immediately when extension load
onDataLoaded callback function will be triggered after ajax load the working hour data. Accessible varialbes: myWorkingHours
onPageReady callback function will be triggered first time once main page ready
onSubPageReady callback function will be triggered everytime sub page ready
